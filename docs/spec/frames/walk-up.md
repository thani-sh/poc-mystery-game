# Walk Up

## Description

Walking up animation - walking away from viewer, left foot forward.

## Pose Details

- Character walking away from the viewer (back view)
- Left foot forward, right foot back
- We see the character's back, not their face
- Arms swinging naturally (right arm forward when left leg forward)
- Body has slight forward lean indicating movement
- Weight on front foot

## Animation Notes

- This is one frame of a 2-frame walk cycle
