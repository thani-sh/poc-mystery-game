# Idle Up - Back Facing

## Description

Character standing idle, facing away from the viewer (back view).

## Pose Details

- Character facing away from viewer
- Standing position, both feet on ground
- Relaxed, neutral stance
- Arms at sides or in a natural resting position
- Weight evenly distributed
- Body centered and upright
- We see the character's back, not their face

## Viewing Angle

- Back view - character faces away from camera
- We see their back fully

## Usage

Default idle pose when character is standing still and facing upward/away on the game map.
