# Idle Down - Front Facing

## Description

Character standing idle, facing toward the viewer (front view).

## Pose Details

- Character facing directly toward viewer
- Standing position, both feet on ground
- Relaxed, neutral stance
- Arms at sides or in a natural resting position
- Weight evenly distributed
- Body centered and upright

## Viewing Angle

- Front view - character faces camera directly
- We see their front fully, not sides or profile

## Usage

Default idle pose when character is standing still and facing downward/forward on the game map.
