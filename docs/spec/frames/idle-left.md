# Idle Left - Side Facing

## Description

Character standing idle, facing to the left (left side view).

## Pose Details

- Character facing left in profile or three-quarter view
- Standing position, both feet on ground
- Relaxed, neutral stance
- Arms at sides or in a natural resting position
- Weight evenly distributed
- Body centered and upright

## Viewing Angle

- Left side view - character faces left
- Profile or three-quarter angle showing the left side

## Usage

Default idle pose when character is standing still and facing left on the game map.
