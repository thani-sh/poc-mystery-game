# Neutral / Idle Expression

## Description

Default expression, calm, eyes forward.

## Facial Features

- Relaxed, neutral expression
- Eyes looking straight ahead or slightly toward viewer
- Mouth closed in a natural, neutral position
- Face completely relaxed, no tension
- Eyebrows in natural, neutral position

## Body Language

- Relaxed posture
- Shoulders level and comfortable
- No hand gestures

## Usage

This is the default portrait used for general dialogue and idle states. It should be the most neutral and versatile expression for the character.
